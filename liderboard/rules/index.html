Règles de calcul du score

Principe général

Le score d’un joueur est calculé en fonction de plusieurs critères dans les différents jeux développé par OneLevel Studio.

Éléments pris en compte

Jumpix :

  • Niveaux terminées : Rapporte dix points par niveaux terminé.
  • Kill : Rapport un point pour chaque ennemi éliminé.

Clic :

  • Nombre d'améliorations acheté : Bonus ajusté pour éviter qu’un élément trop facile à obtenir déséquilibre le score..

Littelgames :

  • Victoire : Un point par victoire à tout les mini-jeux sauf au Tic-tac-toe.

Pawtopia :

  • Mini-Jeux : Des points sont ajoutés après chaques mini-jeu.

Sites :

  • JS Playground : Dix points ajouté pour chaque code exécuté.
  • Convertisseur : Dix points ajouté pour chaque image convertie.
  • ColorToCode : Dix points ajouté pour chaque couleur générée

Autres :

Certains éléments ou actions spécifiques peuvent également apporter des points bonus.

  • Mille points ajouté pour l'instalation de Snake, de JuneOs et de June Net.
  • Un point ajouté pour chaques visites de notre site partenaire Fenyx Esport depuis le lien présent sur notre site.

Formule simplifiée

Le score est calculé en additionnant les points provenant de chaque partie :

Score = Jumpix + Clic + Littelgames + Pawtopia + Sites + Bonus

Jumpix : points des niveaux terminés + kills.
Clic : points des améliorations achetées.
Littelgames : points des victoires.
Pawtopia : points gagnés via les mini-jeux.
Sites : points gagnés via les sites internets.
Bonus : points supplémentaires liés aux actions spéciales et partenariats.

Classement

Le classement est établi en triant les joueurs par score décroissant. En cas d’égalité, d’autres critères peuvent être pris en compte comme le temps ou la date de réalisation.
